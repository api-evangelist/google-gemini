# Google Gemini GraphQL API

Google Gemini is a family of multimodal AI models (Gemini 1.5 Pro, Flash, Ultra). The Gemini API covers text generation, vision, audio, code generation, embeddings, function calling, system instructions, and context caching.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Google Gemini REST/gRPC API
**Documentation:** https://ai.google.dev/docs

- Reference: https://ai.google.dev/api
